Question 2: JavaScript Features

2.1 unshift(), shift(), and split()

unshift(): Adds one or more elements to the beginning of an array.

shift(): Removes the first element from an array.

split(): Splits a string into an array of substrings based on a separator.

Example using CFGdegree streams:


// Original string of CFGdegree streams
let streams = "Web Development,Data Science,Cybersecurity";
// Using split() to create an array
let streamsArray = streams.split(",");
console.log("Streams Array:", streamsArray);
// Using unshift() to add a new stream
streamsArray.unshift("UX/UI Design");
console.log("After unshift:", streamsArray);
// Using shift() to remove the first stream
let removed = streamsArray.shift();
console.log("Removed Stream:", removed);
console.log("Final Streams Array:", streamsArray);

2.2 Object Methods

Object methods are functions defined within an object that can operate on the object's data.

Example with Programming Languages:


let programmingLanguages = {
    languages: ["JavaScript", "Python", "Java"],
    listLanguages: function() {
        console.log("Languages:", this.languages.join(", "));
    },
    addLanguage: function(newLang) {
        this.languages.push(newLang);
        console.log(newLang + " added.");
    }
};
// Using object methods
programmingLanguages.listLanguages();
programmingLanguages.addLanguage("C#");
programmingLanguages.listLanguages();

2.3 DOM Events

onmouseover: Event that triggers when the mouse pointer moves over an element.

Example: Hover over this text to trigger onmouseover event
